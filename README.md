## 📄 프로젝트 소개

👉 영화를 보CINEMA는 영화 트레일러 오픈 API를 활용한 영화 목록 및 조회 사이트입니다.

- 현재상영작, 상영예정작, 인기영화 순위들을 볼 수 있으며
- 아이템 카드들을 클릭 시 디테일 페이지로 넘어가 영화에 대한 자세한 내용들과 미리보기 영상을 감상할 수 있습니다.
- 또한 검색 기능이 가능해서, 영화 제목을 검색 시 원하는 모든 영화 리스트들을 찾아볼 수 있습니다.
